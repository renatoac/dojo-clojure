## src/code_number_to_letter.clj

It gets a string containing numbers separated by #, if you want to include a pause, and result in letters, simulating the behaviour and results from a telephone keypad.

2 = ABC  
3 = DEF  
4 = GHI  
5 = JKL  
6 = MNO  
7 = PQRS  
8 = TUV  
9 = WXYZ
